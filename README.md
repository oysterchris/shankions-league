# Shankions League

**The Most Masculine Game in History**

Football with weapons. Score goals, protect your team, eliminate the opposition.

## Play

Open `index.html` in a browser. No build step required.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move selected player |
| SPACE | Pass / Shoot |
| TAB | Cycle selected player |
| Q | Through ball |
| 1-6 | Select player by number |
| F | Enter combat mode |
| CLICK | Attack (combat) / Select player |
| ESC | Exit combat mode |
| SHIFT | Sprint |

## Concept

Top-down football (FIFA-style) fused with FPS combat (CS-style). Each team has hidden weapon carriers — one rifle, one shank. Win matches to earn cash and upgrade your arsenal.
